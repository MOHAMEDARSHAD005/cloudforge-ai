# Implementation Workflow

Before coding:

1. Read required documents.
2. Produce Task List.
3. Produce Implementation Plan.
4. Produce Files to Create.
5. Produce Files to Modify.
6. Produce Validation Plan.
7. Stop and wait for approval.

During implementation:

- Complete one task at a time.
- Validate after each task.
- Run tests continuously.

After implementation:

- Execute validation checklist.
- Update dynamic docs.
- Produce summary.
